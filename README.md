# Birdswood Secondary School Open Day Presentation

## Project Overview
The Birdswood Secondary School Open Day Presentation is designed to showcase the offerings of the school to prospective students and their families. The presentation is aimed at providing insights into the school's values, achievements, and the educational journey it offers.

## Features
- Interactive slides featuring essential information about the school.
- High-quality images and graphics that reflect the school environment.
- Clean and captivating design for effective communication.

## Slide Structure
1. **Title Slide**: Introduction to Birdswood Secondary School.
   - Contains the school logo and presentation title.

2. **Welcome Note**: A greeting from the principal.
   - Brief message welcoming viewers.

3. **About Us**: Overview of the school's history and mission.
   - Background information and core values.

4. **Programs Offered**: Descriptions of the academic and extracurricular programs.
   - List and brief details of various programs available to students.

5. **Teacher Profiles**: Introduction to key educators and their backgrounds.
   - Photos and short bios of teaching faculty.

6. **Student Testimonials**: Quotes from current students about their experiences.
   - Authentic student voices sharing positive experiences.

7. **Campus Facilities**: Overview of the physical resources available.
   - Images and descriptions of classrooms, libraries, and sports facilities.

8. **Extracurricular Activities**: Information about clubs and sports.
   - Examples of activities students can engage in outside academics.

9. **Community Involvement**: Details on the school's engagement with the local community.
   - Projects and partnerships that showcase community ties.

10. **Parental Involvement**: Information for parents on how they can engage with the school.
   - Opportunities for parents to volunteer and participate.

11. **Success Stories**: Highlighting alumni achievements and contributions.
   - Inspiring stories of what former students have accomplished.

12. **FAQ Section**: Addressing common questions about admissions.
   - Answers to frequently asked questions by prospective students/parents.

13. **Contact Information**: How to reach the school for further inquiries.
   - Contact details and links to the school's social media.

## Image Placement Guide
- Title slide should feature the school logo centered at the top.
- Each program slide can have a relevant image aligned to the right or left.
- Use high-resolution images to ensure clear visuals.

## Setup Instructions
1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/<your-username>/birdswood-open-day-presentation.git
   ```
2. Navigate to the project directory:
   ```bash
   cd birdswood-open-day-presentation
   ```
3. Open the `index.html` file in your preferred browser to view the presentation.

## Navigation Controls
- Use the left and right arrow keys to transition between slides.
- The escape key will exit the presentation mode.

## Customization Guidelines
- To add or modify slides, edit the `slides/` directory files.
- Ensure to maintain consistent styles as defined in `styles.css`.

## Deployment Options to GitHub Pages
1. Push your local changes to GitHub:
   ```bash
   git add .
   git commit -m "Update presentation"  
   git push origin main
   ```
2. Go to your repository settings on GitHub.
3. Scroll down to the "GitHub Pages" section and select the `main` branch as the source.
4. Click "Save" and your presentation will be available at `https://<your-username>.github.io/birdswood-open-day-presentation/`.

Enjoy showcasing the Birdswood Secondary School!
